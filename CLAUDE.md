# マイクラ建築ルーレット

マインクラフト建築アイデアランダム提案アプリ。

## 使い方

### HTML版（メイン）
`index.html` をブラウザで開くだけ。データは `data.js` から読み込む。

### CLI版
```bash
python roulette.py                    # ランダムに1件
python roulette.py -m 和風            # ムードで絞り込み
python roulette.py -s 小 -m のどか    # 複数条件
python roulette.py -l                 # 選択肢一覧
```

## ファイル構成

```
minecraft_build_roulette/
├── index.html       # HTML版アプリ
├── data.js          # HTMLが読み込むデータ（build.pyで生成）
├── builds.csv       # 建築アイデアデータ（原本）
├── build.py         # builds.csv → data.js 変換スクリプト
├── roulette.py      # CLI版アプリ
└── CLAUDE.md        # このファイル
```

## データ構造（builds.csv）

| カラム | 説明 | 選択肢 |
|--------|------|--------|
| name | 建築名 | 自由 |
| category | カテゴリ | 自由 |
| scale | 規模 | 超小 / 小 / 中 / 大 / 超大 |
| mood | ムード | のどか / にぎやか / ダーク / ファンタジー / 和風 / 現代 / SF / 廃墟 / 神秘 / かわいい / レトロ / トロピカル / 冬景色 / スチームパンク / 軍事 / ウェスタン / 北欧 / 中世欧風 / 東南アジア / 地中海 / 砂漠文明 / 中華 / 海賊 / 原始 / ゴシック / 深海 / インド / サイバーパンク / 南米古代文明 / 天空 / ディストピア / 祭り / アフリカ / ネザー / エンド / 沼地 / メサ / キノコ島 / 鍾乳洞 / ディープダーク / 桜の森 |
| difficulty | 難易度 | 1〜5の整数 |
| description | 説明文 | マイクラでの作り方ヒントを含む2〜3文 |
| tags | タグ | パイプ（|）区切りで複数指定。例: `木造\|素朴\|田舎` |

## データを増やす方法

1. 別のClaudeチャットで以下のように頼んでCSVの行を生成し、`builds.csv` に追記する
2. `python build.py` を実行して `data.js` を更新する

**プロンプト例:**
```
以下のCSV形式でマイクラの建築アイデアを20個生成して。
ムード「和風」で規模は小〜大でバラバラに。ヘッダー行は不要。

フォーマット（1行目はサンプル、実際には不要）:
name,category,scale,mood,difficulty,description,tags
農家の納屋,農業,小,のどか,1,干し草ブロックと木材を組み合わせた素朴な納屋。外に農具をアイテムフレームで飾ると雰囲気が出る。,木造|素朴|田舎
```
